HTML

<h1>Title</h1> <h2>Heading 1</h2> This is a <b>bold text</b>. <ul> <li>Some point</li> <li>Some <i>italic</i> point</li> <ul> <li>More points</li> <li>Even more points</li> </ul> </ul> <h2>Heading 2</h2> If you click this <a ref="https://www.google.de/search?q=baby+meme&tbm=isch">link</a> you will see this image:<br> <img src="http://memesvault.com/wp-content/uploads/Funny-Baby-Meme-4.jpg">

WIKI-Syntax - MediaWiki

== Title == === Heading 1 === This is a '''bold text'''. * Some point * Some ''italic'' point ** More points ** Even more points === Heading 2 === If you click this [https://www.google.de/search?q=baby+meme&tbm=isch link] you will see this image: [[http://memesvault.com/wp-content/uploads/Funny-Baby-Meme-4.jpg]]

WIKI-Syntax - DokuWiki

====== Title ====== ===== Heading 1 ===== This is a **bold text**. * Some point * Some //italic// point * More points * Even more points ===== Heading 2 ===== If you click this [[https://www.google.de/search?q=baby+meme&tbm=isch|link]] you will see this image: {{http://memesvault.com/wp-content/uploads/Funny-Baby-Meme-4.jpg}}

reStructuredText

===== Title ===== Heading 1 --------- This is a **bold text**. - Some point - Some *italic* point - More points - Even more points Heading 2 --------- If you click this `link <https://www.google.de/search?q=baby+meme&tbm=isch>`_ you will see this image: .. image:: http://memesvault.com/wp-content/uploads/Funny-Baby-Meme-4.jpg

Markdown

Title ===== Heading 1 --------- This is a **bold text**. * Some point * Some *italic* point * More points * Even more points Heading 2 --------- If you click this [link](https://www.google.de/search?q=baby+meme&tbm=isch) you will see this image: ![image](http://memesvault.com/wp-content/uploads/Funny-Baby-Meme-4.jpg)

LaTeX

\documentclass{article} \usepackage{hyperref} \usepackage{graphicx} \begin{document} \title{Title} \maketitle \section{Heading 1} This is a \textbf{bold text}. \begin{itemize} \item Some point \item Some \textit{italic} point \begin{itemize} \item More points \item Even more points \end{itemize} \end{itemize} \section{Heading 2} If you click this \href{https://www.google.de/search?q=baby+meme&tbm=isch}{link} you will see this image: \includegraphics{http://memesvault.com/wp-content/uploads/Funny-Baby-Meme-4.jpg} \end{document}
